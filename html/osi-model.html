OSI Model Overview

What is the OSI Model?

The Open Systems Interconnection (OSI) model is a conceptual framework used to describe the functions of a networking system. It divides network communication into seven distinct layers, from the physical transmission of data to the application-level interactions.

Layer 1: Physical

Deals with the physical transmission of raw bits over a medium (cables, fibers, radio signals).

Layer 2: Data Link

Responsible for node-to-node data transfer and handles error detection and correction.

Layer 3: Network

Manages routing and forwarding of data packets across different networks (e.g., IP addresses).

Layer 4: Transport

Ensures complete data transfer with flow control and error recovery (e.g., TCP and UDP).

Layer 5: Session

Establishes, manages, and terminates sessions between end-user applications.

Layer 6: Presentation

Translates and formats data for the application layer, handling encryption and compression.

Layer 7: Application

Provides network services to end-user applications (e.g., HTTP, SMTP, FTP).
